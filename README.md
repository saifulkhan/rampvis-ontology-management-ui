# UI

## Quick start

Key dependencies:

- Node version 12.17.0
```bash
npm install -g typescript@3.8.3
npm install -g @angular/cli@9.1.7
```

Start development instance:
```
npm install
npm start
```

Navigate to [localhost:4200](localhost:4200)
